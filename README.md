# Multilabel Cyberbullying Detection

Multilabel classification of cyberbullying in **Bangla text** using BERT-based transformer models.

## Overview
Detects 5 categories of cyberbullying — **bully, sexual, religious, threat, spam** — from Bangla social media comments using fine-tuned BERT models.

## Features
- Bangla text preprocessing and cleaning
- BERT-based multilabel classification
- Optimizer comparison: RAdam, RMSprop, Adafactor
- Data balancing: upsampling and downsampling
- Evaluation: F1-score, Hamming Loss, Precision, Recall, ROC-AUC

## Tools & Libraries
Python, PyTorch, HuggingFace Transformers, scikit-learn, pandas, Jupyter Notebook
